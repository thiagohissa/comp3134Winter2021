1) What is the definition of server port?
A server port is a connection point through which information flows from the server to a computer and vice versa. 

2) What is server port range?
Ports are represented by numbers and port numbers range from 0 to 65535, but only 0 to 1023 (1024 ports) are designated as well-known ports. Also, some port numbers are reserved to specific usability (e.g. 80 is used by web servers)

3) How many bits does number above represent?
1024 is 10 bits (2^10 = 1024)
